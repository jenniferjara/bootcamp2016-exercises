#PAR - IMPAR
*Este repositorio contiene el proceso que le permite al usuario ingresar un número y mostrar si es par o impar y se describe en el siguiente pseodocódigo. A continuación se puede ver en una imagen el diagrama de flujo del ejercicio descrito líneas arriba.*

##Psedocódigo
1. Inicio. 
2. Definir la variable: numero.
3. Si numero cumple la condición es par.
4. De lo contrario es impar.
5. Fin.
